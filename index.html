نبذة عن المشروع

💡 الفكرة والهدف

لاحظت عائشة أن مدرسة ذي قار للتعليم الأساسي تعاني من مشكلة متكررة كل عام دراسي: إنشاء مجموعات واتساب جديدة لكل صف وشعبة، مما يُشتّت إدارة المدرسة والأمهات بين عشرات المجموعات ويجعل التواصل فوضوياً وغير منظّم.

جاء مشروع بوابة التواصل المدرسي حلاً رقمياً موحداً يستبدل مجموعات الواتساب المتشعبة بمنصة إلكترونية واحدة، تُتيح لأولياء الأمور إرسال استفساراتهم مباشرة للإدارة، كما تنشر الإدارة الإعلانات والفعاليات التي تظهر فوراً لجميع الزوار من أي جهاز.

Implementation Steps

خطوات التنفيذ

١
🔍 تحديد المشكلة

رصدت عائشة مشكلة تتكرر كل عام دراسي: إنشاء مجموعات واتساب جديدة لكل صف وشعبة، مما يُشتّت إدارة المدرسة والأمهات بين عشرات المجموعات ويُصعّب التواصل المنظّم ومتابعة الإعلانات المدرسية.

📊 تحليل المشكلة
٢
✏️ التخطيط والتصميم الأولي

رسم النماذج الأولية (Wireframes) للصفحتين الرئيسيتين: بوابة أولياء الأمور ولوحة تحكم المعلم، وتحديد الوظائف المطلوبة لكل قسم.

🎨 تصميم UX/UI
٣
⚙️ إعداد قاعدة البيانات

إنشاء Google Sheets بأربع أوراق عمل (الرسائل، الإعلانات، الفعاليات، الإحصائيات) وربطها بـ Google Apps Script لاستقبال البيانات ومعالجتها.

🗄️ قاعدة البيانات
٤
💻 بناء الصفحات

تطوير صفحة أولياء الأمور (نموذج التواصل، الإعلانات، الفعاليات) ولوحة تحكم المعلم (إدارة الرسائل، نشر الإعلانات، عرض الإحصائيات) باستخدام HTML وCSS وJavaScript.

🌐 تطوير الويب
٥
🔗 الربط والاختبار

ربط الصفحتين بـ Google Apps Script، واختبار جميع الوظائف (إرسال الرسائل، نشر الإعلانات، تحديث الحالات) والتأكد من ظهور البيانات فوراً لجميع الزوار.

🧪 الاختبار والتحقق
٦
🚀 النشر والتوثيق

نشر البوابة على GitHub Pages لتكون متاحة للجميع مجاناً بدون خادم، وإعداد توثيق شامل للمشروع يشمل دليل الاستخدام وهوية المشروع البصرية.
